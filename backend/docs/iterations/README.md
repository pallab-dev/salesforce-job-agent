# Iteration Notes

Use this folder for detailed milestone notes when a change is too large for a short `CHANGELOG.md` entry.

## Suggested file naming

- `001-modular-refactor.md`
- `002-multi-profile-actions.md`
- `003-config-driven-profiles-remotive.md`

## Suggested template

```md
# <Iteration Title>

## Goal

What problem this iteration solves.

## Changes

- Code changes
- Workflow changes
- Config changes

## Commits

- `<short_sha>` - commit message

## Risks / Known Gaps

- ...

## Next Steps

- ...
```
